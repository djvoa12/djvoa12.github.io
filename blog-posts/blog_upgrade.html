For week nine HTML/CSS challenge, I chose to upgrade my personal blog website.  I haven't touched the HTML/CSS code since the end of week two so I figured this would be a wonderful opportunity to revamp my blog pages.  To facilitate this, I asked couple tach-savvy friends to review and provide feedback for my blog page.  They suggested that I may want to remove text-decoration from the navigation buttons, go with minimalistic approach, remove certain pages, and center all the elements to make it more fluid.  See my changes below.

Main Page/Blog/Projects:
- Fixed footer to include svg images of my GitHub, Linkedin, Soundcloud, and Twitter accounts.
- Centered everything to make it more fluid and responsive.
- Made my profile image oval-shaped (found the CSS code online)
- Removed 'Education' and Work Experience' pages as my Linkedin profile provides in-depth information on both subjects.
- Switched and transposed navigation bar from left-hand side to the top.
- Removed text-decoration on the navigation links.
- Classified blog entries into three types: technical, cultural, and miscellaneous.
- Renamed all blogs to match their title.

Blog-Template:
- Changed content background to parchment color to better visibility of content.
- Used 'pre' tags to show content as is from HTML code.
- Used syntax highlighter code to display code in various languages.
