A class is just a way of organizing and producing objects with similar attributes and methods. A basic class consists of the 'class' keyword and the name of the class.

class RealEstate
end

You need to create 'initialize' method, which acts as a function that "boots up" each object the class creates.

class RealEstate
def initialize
end
end

We now have to ensure that the class above has '@home_value'. '@' before a variable is used to tell that it's an instance variable, which means that the variable is attached to the instance of the class. You are now ready to start creating objects.

class RealEstate
def initialize(home_value)
@home_value = home_value
end
end

home = RealEstate.new(178484)

Let's create a method 'calc_commission' within the class that returns your estimated commission payment to your realtor if you were to sell the home. We are assuming that the commission rate nationwide is about 6%. You can now apply the new method to the home object and it will return your estimated commission payment.

class RealEstate

def initialize(home_value)
@home_value = home_value
end

def calc_commission
(@home_value * 0.06).round(2)
end
end

home = RealEstate.new(178484)
home.calc_commission #--> 10709.04
